### Founder / Team Photography Style Guide

When commissioning or selecting founder photos:
- **Background:** Dark (charcoal, dark stone, or dark brick)
- **Lighting:** Warm key light from one side; slight amber/orange cast preferred
- **Mood:** Thoughtful, focussed — workshop or home-office setting
- **Avoid:** Pure white studio backgrounds, over-processed HDR, corporate blue tones

## 7. Prompt Modifiers Reference

### Core Style Modifiers (use in all Foundry Apps prompts)

```
dark background, warm amber and orange lighting, moody atmosphere,
professional editorial photography or digital art, no text in image
```

### Quality Boosters (Midjourney v6)

```
--style raw          # More photorealistic, less AI-processed look
--v 6                # Midjourney v6 (current best)
--q 2                # Higher quality (slower)
```

### Aspect Ratio Quick Reference

| Use Case | Midjourney Flag | Pixels |
|----------|----------------|--------|
| Hero / landing page | `--ar 16:9` | 1920×1080 |
| OG / social share | `--ar 1200:630` | 1200×630px |
| Square (social) | `--ar 1:1` | 1080×1080 |
| Blog header | `--ar 3:1` | 1500×500 |
| Portrait / mobile | `--ar 9:19.5` | 1170×2532 |
| Feature graphic | `--ar 4:3` | 1200×900 |
| Twitter banner | `--ar 3:1` | 1500×500 |

### Negative Prompts (things to exclude)

Add `--no` followed by:
```
--no white background, stock photography, people, text, watermarks,
cartoons, anime, flat illustration, overly saturated colours,
blue corporate aesthetic, office scenes
```

### DALL-E 3 Style Suffix

Append to any DALL-E 3 prompt:
```
Photorealistic. Dark background with warm amber/orange accent lighting.
No text or lettering visible in the image. Professional editorial style.
```

### Stable Diffusion XL Negative Prompt

```
negative_prompt: (text:1.4), (watermark:1.4), (logo:1.2), white background,
bright background, stock photo, corporate, cheerful, saturated, flat,
cartoon, anime, blurry, low quality, nsfw
```
